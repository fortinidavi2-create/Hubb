COMO USAR:
===========

1. Com QEMU:
   qemu-system-x86_64 -cdrom HubbOS.iso -m 1024 -boot d

2. Com VirtualBox:
   - Crie VM Linux 64-bit
   - Aponte para HubbOS.iso no CD-ROM
   - Inicie a máquina

3. Com USB (físico):
   sudo dd if=HubbOS.iso of=/dev/sdX bs=4M
